# SPA para compartir (Ligth novels, web novels)
El objetivo de esta aplicación es compartir traducciones
sin anuncios y siempre mostrando los créditos a sus respectivos creadores.
No se pretende lucrar con esta aplicación.
si desea apoyar puede unirse a nuestro grupo
y ofrecerse para un rol, más adelante se estarán presentando nuevos roles.
¡No importa si no tienes experiencia en algunos de estos roles, ¡podemos ayudarte!
¡Gracias por su apoyo!

## Guia de instalacion
### Instalacion de dependencias
- [git](https://git-scm.com/install/)
- [node](https://nodejs.org/en/download/)

### Clone este repositorio
1. clone el repositorio en una carpeta de su preferencia
```
git clone https://github.com/Guchangge28/roolkan.git
```
2. entrar en la carpeta del repositorio
```
cd roolkan
```
3. instalar paquetes
```
npm install
```
4. iniciar la aplicación en modo desarrollo
```
npm run dev
```
